# CSS进阶应用

> CSS本身有复杂重复和不同浏览器核心不同语法的问题。所以现在有一些扩展框架让CSS更容易，如SASS和LESS。 SASS是基于Ruby，而LESS基于Javascript。 据说SASS更为完备和成熟，但是需要学习一门新语言。因此，我们的重点还是放在LESS。
